My Projects

Danish Blooddonor Database

A web application with the goal of replacing an existing MS Access database which allows researchers and lab personnel create, track, and manage all blood and plasma samples used for research across Denmark.

  • Built using Flask, Angular / AngularJS, Postgres
  • The system currently handles 2.5 million active blood/plasma samples used for research purposes.
  • Speed and performance are essential, as lab personnel are scanning hundreds of samples per minute; all sample IDs in this system must adhere to strict rules depending on the type of sample, their location/region, and must be unique in the system.
  • Historic data about when and where samples were scanned, shipped/received, and aliquoted for research is also an important requirement of the system, so all transactions in the database are logged to provide the relevant data for audits.

FOREVER

A project in collaboration between Københavns Universitet and Synoptik, creating a centalized web application for optometrists to evaluate and diagnose eye exams of patients.

  • Built using Django, Angular / AngularJS, Postgres.
  • The system handles and displays all data as well as Dicom, OCT, and perimetry images for eye exams conducted on patients in the FOREVER study.
  • The goal is to allow optometrists to evaluate these examinations separately from Synoptik, to see if the evaluation conclusions from Synoptik and the optometrists are aligned, thus allowing for more accurate diagnoses of overall eye health of patients.

INNODIA

A project aimed at providing a platform for medical professionals across the EU to capture clinical trial data about participants suffering from type 1 diabetes.

  • Built using Flask, AngularJS, Postgres.
  • INNODIA is an eCRF, clinical trial platform aimed at evaluating clinical data from individuals with newly diagnosed type 1 diabetes as well as those who are predisposed to developing T1D.
